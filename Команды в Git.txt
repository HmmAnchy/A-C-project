1 Сначала клонируем наш репозиторий:
git clone <ссылка на репозиторий> - ссыкла появится, если нажать на кнопку "<>code" (HTTPS)

2 Дальше:
-git config --global user.name "Имя";
-git config --global user.email "Почта";

3 Потом синхрон. другие ветки (т.е. Testing-1), чтобы они были доступны нам на лок. репоз.
git fetch origin
git merge otigin/<Название ветки>
git pull

4 Переходим на ветку Testing-1(на самом деле ветка не появится после 3 шага, появится как только перейдем на ветку):
git checkout Testting-1 (проверяем: git branch)

5 Работаем на ветке Testing-1, чтобы не засорять main, а также коммитим на ветке Testing-1

6 Основные команды
git log - выводит коммиты
git status - выводит файлы, которые были изменены или созданы ( чтобы закоммить изменения, сначала добавляем в индекс (git add .), т.е они становятся зеленными)
git commit -m "<коммент>" - создание коммита

7 Синхронизация с лок. репоз. на удаленный
git push
